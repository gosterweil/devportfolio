Projects

VI2SQL

A ruby script I wrote for Hofstra CS to automate data gathering on VMWare VMs. It uses vSphere web services API to pull VM properties and write them to a MysQL database.

LDAP Sync

A couple of perl scripts which sync data between MySQL and Hofstra's LDAP AD servers, ensuring both sources have the data we need

vSphere utils

A collection of several Ruby scripts I made to automate VM maintenance and cleanup tasks.

They're written in a UNIX style, with each tool implementing a single task in a way that can be easily piped together

The lsvi script writes properties such as VM owners from vSphere to STDOUT in configurable format

hofad takes STDIN input from lsvi and checks AD, outputting user info like ID, name, and if the account is disabled

vm_batch takes VM paths as input and allows us to quickly power off, move or delete large numbers of VMs from (e.g.) disabled accounts

IP Block Update

A Python script that tracks IP range availability for VM use.

It marks input IP ranges as assigned or free in MySQ so we know what IPs can be used.

It splits or merges rows in MySQL as needed, allowing flexibility in assignments
